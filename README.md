#HELLO-RAILS-BACK-END

This project is an API that returns random greetings renders random greetings.

## Build With

- Ruby
- Ruby on Rails
- PostgreSQL


## Installation
```
  $ git clone git@github.com:SimonGrchevski/hello-rails-back-end.git

  $ cd hello-rails-back-end

  $ bundle

  $ rails db:create

  $ rails db:migrate

  $ rails db:seed

  $ rails s
  
```
